## Description

ブログ記事のmetaタグを充実させ、SEOとSNSシェア時の見栄えを改善する。

現状、ZolaテンプレートにdescriptionやOGP関連のmetaタグが入っていない。
front matterの`description`フィールドから以下のタグを自動生成する仕組みをテンプレートに組み込む：

- `<meta name="description">` — SEO用（検索結果スニペット）
- `og:title`, `og:description`, `og:image`, `og:type`, `og:url` — SNSシェア時のカード表示
- `twitter:card`, `twitter:title`, `twitter:description` — X（Twitter）でのカード表示

## Related Issues

- `20260220-0207` ブログの見た目改善（テーマ・CSS・レイアウト） (related to)

## Approach

1. 現状のテンプレート（`templates/base.html`等）の`<head>`セクションを確認
2. front matterに`description`フィールドを追加する規約を決める
3. Tera テンプレートで`page.description`や`page.extra`からmetaタグを自動出力するブロックを実装
4. 既存記事にdescriptionを追加（優先度の高い記事から段階的に）
